# Virtual Home Lab
This repository displays a step by step process on how I set up a virtual home lab.
PS. Please refer to the Document "Building a Virtual IT Lab at Home - Step-by-Step Documentation", for the steps

## Purpose
The home lab is designed to:
- Learn how virtual machines work
- Practice installing and configuring a windows OS in a virtual environment.
- Build confidence working with virtualisation tools

## Tools & Environment
Virtulisation Software - VirtualBox
Guest Operating System - Windows 8
Host Machine - Personal Computer

Project is for educational and learning purposes only!
